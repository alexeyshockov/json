# Json

Json wrapper to provide a more robust API matching PHP 7.3 with exceptions rather than errors.

The following json options are always used

- JSON_UNESCAPED_UNICODE
- JSON_BIGINT_AS_STRING
- JSON_THROW_ON_ERROR
